Sobre o Projeto

O Radar de Atrasos é uma aplicação web desenvolvida por Jessé Melo, cientista da computação, como projeto aplicado de Machine Learning no contexto do Oracle Next Education. O sistema utiliza técnicas de análise preditiva para estimar a probabilidade de atraso de voos, incorporando métodos de explicabilidade (SHAP) para interpretar os fatores que impactam o resultado.

A aplicação não armazena dados pessoais dos usuários, funcionando apenas como ferramenta de simulação e análise preditiva educacional.
