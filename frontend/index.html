Sobre o Projeto

O Radar de Atrasos é uma aplicação de Machine Learning que estima a probabilidade de atraso em voos utilizando dados públicos da ANAC.

O projeto utiliza regressão logística e XGBoost para modelagem, além de SHAP para explicabilidade das previsões.
